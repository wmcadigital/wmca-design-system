# Autocomplete pattern

This package contains the Autocomplete pattern styles and template used by the WMCA design system.

Files:

- `dist/_autocomplete.scss` - distributable SCSS for the pattern
- `dist/_autocomplete.njk` - Nunjucks template example
- `src/_autocomplete.scss` - source SCSS
- `src/_autocomplete.njk` - source Nunjucks template

Build:

```bash
npm run build
```
