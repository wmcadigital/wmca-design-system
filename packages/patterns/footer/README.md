# Footer pattern

This package contains the footer pattern used across WMCA sites.

Build: `npm run build` from within the package folder.

Outputs are placed in `dist/` and include compiled CSS and rendered HTML examples.
# Footer (pattern package)

This package contains the Footer pattern templates, styles and example JS.

Build (copies `src/` → `dist/`):

```bash
npm --prefix packages/patterns/footer run build
ls -R packages/patterns/footer/dist
```
